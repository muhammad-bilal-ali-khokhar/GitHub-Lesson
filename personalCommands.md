                            PERSONALY USE 
For Check Out 
git status

for add all changed
git add .

for commit
git commit -m "commit"

for push all 
git push 

for push latest change 
git pull


for create branch 
git branch branch name 


for create and goto created branch
git branch -b branchName 


for push to specific branch 
git pull --set-upstream origin branchName


for merge any file
git merge branchName
